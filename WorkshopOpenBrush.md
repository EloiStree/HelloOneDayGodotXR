S’il y a le moindre problème ou bug, vous pouvez laisser enfoncé 3 à 7 secondes jusqu’à entendre un “toudoum”.   
Ça redémarre le casque.   
Un simple clic permet de lui demander de se réveiller.

Celui de gauche : menu pour les jeux   
* Permet d’accéder à certains menus en jeu.   

Celui de droite : menu pour le casque Meta   
* Laisser enfoncé pour recentrer les jeux ou l’application.   
* Cliquer une fois permet de sortir et de revenir dans l’application.  

Pour le menu de droite, qui est très important, il y a un équivalent avec votre main, car le casque est utilisable sans les manettes.   

Il faut pincer avec les doigts vers le haut :  
[<img width="868" height="488" alt="image" src="https://github.com/user-attachments/assets/2680b49a-159a-4d65-b05b-bc0533dadf28" />](https://youtu.be/tweBgb79VyE?t=240)
[https://youtu.be/tweBgb79VyE?t=240](https://youtu.be/tweBgb79VyE?t=240)

* Pincement rapide : sortir et rentrer dans l’application
* Pincement long : recentrer les jeux ou applications

Cliquer sur l’horloge : paramètres rapides du casque
Rouge (caméra) : enregistrer ce que l’on fait
Bleu (navigateur) : navigateur Meta
Orange (Store) : permet d’acheter et télécharger de nouveaux jeux
À droite de la petite barre : applications ouvertes récemment
9 points : lancer les applications installées sur le casque

Cliquer sur les 9 petits points : cela ouvre la bibliothèque
